# Nível 2: todos os modelos anotam o mesmo texto em paralelo
async def annotate_parallel(text: str):
    tasks = [
        model.ainvoke(text) # HTTP async → Ollama
        for model in self.models
    ]
    # Aguarda TODOS simultaneamente
    results = await asyncio.gather(*tasks)
    return results # [pred_deepseek, pred_qwen, pred_gemma, pred_mistral, pred_llama]

⚡ Performance: Sequencial vs Paralelo (Async)
🐢 SEQUENCIAL
Um modelo de cada vez — aguarda resposta antes do próximo
~35s por texto (5 modelos)
500 textos ≈ 4h 51min
🚀 PARALELO (ASYNC)
Todos os modelos chamados via asyncio.gather() — Ollama serve em paralelo
~7s por texto
500 textos ≈ 58 minutos
